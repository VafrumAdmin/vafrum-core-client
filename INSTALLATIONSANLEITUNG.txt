============================================================
       VAFRUM CORE - Installationsanleitung
============================================================

Vafrum Core verbindet Ihre Bambu Lab 3D-Drucker mit der
Vafrum Cloud-Plattform. So koennen Sie Ihre Drucker von
ueberall ueberwachen und steuern.

------------------------------------------------------------
  VORAUSSETZUNGEN
------------------------------------------------------------

- Windows 10+, macOS 12+ oder Linux (64-Bit)
- Bambu Lab Drucker im lokalen Netzwerk
- Vafrum-Account mit API Key (erhalten Sie von Ihrem Admin)

------------------------------------------------------------
  INSTALLATION
------------------------------------------------------------

--- WINDOWS ---

1. Starten Sie "VafrumCore-Setup-*.exe"
2. Folgen Sie dem Installationsassistenten
3. Nach der Installation startet Vafrum Core automatisch

--- macOS ---

1. Entpacken Sie die ZIP-Datei
2. Verschieben Sie "Vafrum Core.app" in den Programme-Ordner
3. Beim ersten Oeffnen erscheint die Meldung:
   "Vafrum Core nicht geoeffnet" -> Klicken Sie auf "Fertig"
4. Oeffnen Sie: Systemeinstellungen -> Datenschutz & Sicherheit
5. Dort steht "Vafrum Core wurde blockiert"
   -> Klicken Sie auf "Dennoch oeffnen"
6. Im naechsten Dialog nochmals "Dennoch oeffnen" bestaetigen
7. Ab jetzt startet die App ganz normal

   Hinweis: Die Software befindet sich in der Beta-Phase und
   ist noch nicht mit einem Apple-Zertifikat signiert. Die
   Sicherheitswarnung ist normal und unbedenklich.

--- LINUX ---

1. Entpacken Sie die ZIP-Datei
2. Machen Sie die App ausfuehrbar: chmod +x VafrumCore
3. Starten Sie mit: ./VafrumCore

------------------------------------------------------------
  EINRICHTUNG
------------------------------------------------------------

1. Geben Sie Ihren API Key ein (beginnt mit "vfk_...")
   -> Den API Key erhalten Sie von Ihrem Vafrum-Administrator

2. Klicken Sie auf "Verbinden"

3. Ihre Drucker erscheinen automatisch in der Liste

Das war's! Vafrum Core laeuft im Hintergrund und:
- Verbindet sich automatisch beim Windows-Start
- Stellt Kamera-Streams bereit
- Leitet Druckerbefehle weiter
- Aktualisiert sich automatisch

------------------------------------------------------------
  HAEUFIGE FRAGEN
------------------------------------------------------------

F: Wo bekomme ich meinen API Key?
A: Ihr Vafrum-Administrator erstellt einen API Key fuer Sie
   unter https://vafrum-core.de im Admin-Bereich.

F: Meine Drucker erscheinen nicht?
A: Stellen Sie sicher, dass die Drucker eingeschaltet und
   im selben Netzwerk wie dieser PC sind. Die Drucker
   muessen in Ihrem Vafrum-Account konfiguriert sein.

F: Wie aktualisiere ich die App?
A: Updates werden automatisch heruntergeladen. Sie sehen
   einen gruenen Banner oben in der App wenn ein Update
   bereit ist. Klicken Sie auf "Jetzt installieren".

F: Kann ich die App auf mehreren PCs installieren?
A: Ja, aber pro Standort/Netzwerk reicht eine Installation.
   Die Bridge muss im selben Netzwerk wie die Drucker sein.

F: Die App zeigt "Auth Fehler"?
A: Pruefen Sie ob der API Key korrekt ist. Kopieren Sie
   ihn erneut aus dem Vafrum Admin-Bereich.
